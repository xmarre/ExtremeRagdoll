GUIDED ARROW v1.0.3
Target: Mount & Blade II: Bannerlord 1.3.15

INSTALLATION
1. Delete any existing Modules/GuidedArrow folder.
2. Extract the GuidedArrow folder into the Bannerlord Modules directory.
3. Enable Guided Arrow in the launcher.
4. Configure camera mode and offsets in MCM.

CONTROLS
- Move mouse: steer the guided arrow or bolt.
- Q / E: decrease / increase guidance time speed.
- Escape or right mouse: cancel and return to the player.

PROJECTILE CAMERAS
- Camera Mode 0: First Person. The camera sits immediately behind the projectile and looks exactly along the current flight path.
- Camera Mode 1: Third Person. The chase rig maintains a fixed distance and elevation angle in the projectile's local frame, so it rotates with every change in arrow direction.
- Optional centered crosshair support is available in MCM.

NOTES
- Guidance applies only to player-fired arrows and bolts in missions.
- No campaign-map scanning or campaign ticks are used.
- Kill cinematics trigger only for confirmed kills by the tracked guided projectile.
- RTS Camera custom-camera ownership is preserved and restored after the return transition.
